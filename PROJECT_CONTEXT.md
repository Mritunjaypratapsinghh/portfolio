# GitHub Portfolio

## Tech Stack
- Language: Python, HTML/CSS/JS
- Framework: FastAPI (backend)
- Database: None
- Queue/Cache: None
- Cloud Services: GitHub Pages (frontend), Render (backend)

## Architecture Overview
Static portfolio site with a FastAPI backend for AI chat (Groq) and resume generation (WeasyPrint).

## Deployments
- Frontend: https://mritunjaypratapsinghh.github.io/portfolio/
- Backend: https://portfolio-api-8j7b.onrender.com
- Chat: https://mritunjaypratapsinghh.github.io/portfolio/chat.html

## Common Commands
- Run backend: `source venv/bin/activate && uvicorn main:app --reload`
- Deploy frontend: Push to `main` branch (GitHub Actions)
